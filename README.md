# 🌦️ Weather App

🚀 A simple and interactive Weather Application built using HTML, CSS, and JavaScript.  
This project uses a weather API to fetch real-time weather data for any city entered by the user.

---

## 📌 Overview
This project allows users to search for any city and get live weather updates such as temperature, humidity, and weather conditions.

It is built to strengthen core web development and API integration skills such as:
- Fetching data from APIs
- Handling JSON responses
- DOM manipulation
- Event handling
- Dynamic UI updates

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- OpenWeather API (or any weather API used)

---

## ✨ Features
- 🌍 Search weather by city name
- 🌡️ Displays temperature in real time
- 💧 Shows humidity and weather conditions
- 🌤️ Dynamic weather icons (if added)
- ⚡ Instant API response updates
- 📱 Simple and responsive UI
